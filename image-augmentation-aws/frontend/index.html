📋 How it works:

  1. Upload: Select and upload your image (JPG, JPEG, or PNG)
  2. Auto-resize: Image is automatically resized to 256×256 pixels
  3. Rotate: 4 versions are created with different rotations
  4. Store: All images are organized in AWS S3 by rotation angle
  5. Download: Get your processed images instantly
